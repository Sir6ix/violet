## 📦 Split Archives

If a folder contains:

```txt
file.zip
file.z01
file.z02
```

You must:

```diff
+ Download ALL parts
+ Keep them in the SAME folder
+ Open ONLY the .zip file
+ Extract using WinRAR / 7-Zip
```

The extraction software will automatically combine all parts.
